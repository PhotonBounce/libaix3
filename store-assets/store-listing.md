## Full Description (max 4000 chars)
Stay ahead of threats with OpsBrief — your personal AI-powered security intelligence agent.

OpsBrief delivers daily, curated briefings straight from the NVD (National Vulnerability Database), GitHub Security Advisories, and Cisco security feeds — personalized precisely to your IT infrastructure. No more scrolling through irrelevant CVEs. We surface what matters to your stack, rank it by severity, and let you dig deeper with an AI chat assistant.

**What you get:**
- **Daily AI Briefings**: Personalized security intel every morning based on your tech stack — servers, frameworks, libraries, and cloud services.
- **CVE Intelligence**: Direct integration with NVD, GitHub, and Cisco. Every CVE includes CVSS scores, affected versions, patches, and references.
- **AI Chat Follow-Up**: Ask questions about any vulnerability. Compare CVEs. Get remediation advice. Powered by OpenAI.
- **Saved Intel**: Bookmark critical CVEs and conversations. Build your own security knowledge base.
- **PWA + Android**: Install as a web app on desktop, or download the Capacitor-wrapped native Android app from Google Play.

**Built for IT Professionals:**
System administrators, DevOps engineers, security analysts, SREs, and network engineers who need fast, reliable vulnerability intelligence without the noise.

**Key features:**
- Personalized daily CVE briefings
- Natural language AI chat for threat analysis
- Save and organize critical security intel
- Admin dashboard with user analytics
- Dark theme by default, offline support, and push notifications planned for v1.1
- FastAPI backend with enterprise-grade security

OpsBrief doesn't just inform — it empowers you to act before threats do.

Download now and make your first vulnerability briefing personal.
